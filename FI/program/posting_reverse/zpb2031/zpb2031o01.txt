MODULE modify_0100 OUTPUT.

  IF so_glacc-low IS INITIAL AND so_glacc-high IS INITIAL.

    LOOP AT SCREEN.
      IF screen-name = 'ZSPB3300-GLACC'.
        screen-active = 0.

      ELSEIF screen-name = 'ZSPB3300-GNAME'.
        screen-active = 0.
      ENDIF.

      MODIFY SCREEN.

    ENDLOOP.

  ENDIF.
